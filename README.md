# my studymate💻📖

<pre>
<code>
공부에 더욱 집중하고 시간관리를 도와주는 프로젝트입니다.
뽀모도로 기능과 to do list 기능을 이용하여 공부 스케줄 관리를 쉽게할 수 있습니다.
